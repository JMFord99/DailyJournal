The Daily Journal app is designed so that users can choose whichever day in the calendar year and create a journal entry for that day. It’s possible to not only create journal entries for the current day, but also entries for past and future days, making the app very diverse in its application. The user can decide that they wanted to right about their day yesterday, or to write about what they want their day to be like tomorrow. There are also alarms that can be set to notify the user that their entry for that day has not been submitted. When the user opens the app, they are greeted by two buttons, one to complete today’s prompt, and another to view past prompts. When the user chooses to create today’s prompt, they are given a large amount of space to write text, in addition to having a slider for the user’s mood, and being able to make certain entries favorites. The user can also title the entry, that way, they could choose what to write about for each day. The goal is to make the journal entries as customizable as possible so that each use can have the unique experience that they want. This app was created by Daniel Petruso, Elizabeth Jordan, Jeremy Ford, and Niannian Liang
